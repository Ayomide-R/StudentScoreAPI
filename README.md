# Student Score API #
**API desenvolvida para ajudar estudantes a descobrirem suas médias e suas notas necessárias para passar em matérias.**

OBS:
Banco de dados, estou fazendo uma relação de um para muitos. Um aluno pode ter várias matérias mas uma matéria só pode pertencer a um aluno. Estou fazendo a regra de negocio desta forma para que os alunos possam criar o modelo de matérias que querem calcular, para fazer algo individual e não compartilhado.
